## Answer: Resource Library Page (Resource Manager)

When mentors/admins share resources with mentees, the mentees will see them in the **Resource Library** page.

## For Mentees:

### How to Access Shared Resources:

1. Go to **Resource Library** (or Resources page)
2. Click the **"Shared With Me"** button at the top
3. See all resources that mentors have shared with you
4. Each resource shows **"Shared by [Mentor Name]"** underneath

### What Mentees Can Do:

âœ… **View** all resources shared with them
âœ… **Download** shared resources
âœ… **Preview** PDFs, videos, and images
âœ… **See who shared it** (mentor/admin name displayed)
âœ… **Access public resources** (in "All Resources" tab)

### What Mentees CANNOT Do:

âŒ **Upload** new resources (only mentors/admins can upload)
âŒ **Delete** resources (only owners and admins can delete)
âŒ **Share** resources with others (only mentors/admins can share)

## For Mentors/Admins:

### After Sharing Resources:

1. Resources are **immediately available** to mentees
2. Mentees see them in **"Shared With Me"** tab
3. Each shared resource shows **"Shared by [Your Name]"**
4. No notification is sent (future enhancement)

## Resource Library Tabs:

### 1. **All Resources** (Everyone)

- All public resources uploaded by anyone
- Available to all users (mentees, mentors, admins)

### 2. **My Resources** (Mentors & Admins only)

- Resources uploaded by the current user
- Only visible to mentors and admins

### 3. **Shared With Me** (Mentees only) â­ NEW!

- Resources specifically shared with the mentee by their mentor
- Shows mentor's name on each resource
- Helps mentees find curated content from their mentor
